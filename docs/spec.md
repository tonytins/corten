# Specifications

## Instruction Set

| Code | Operation  | Comment |
| --- | --- | --- |
| 0 | LOAD | Load program |
| 1 | ADD | Add |
| 2 | SUB | Subtract
| 3 | MUL | Multiply |
| 4 | DIV | Division |
| 5 | HLT | Halt |
| 6 | JMP | Jump |
| 7 | JMPF | Jump forward |
| 8 | JMPB | Jump backward |
| 9 | EQ | Equal |
| 10 | NEQ | Not equal |
| 11 | GTE | Greater then or equal to |
| 12 | GT | Greater then |
| 13 | LTE | Less then or equal to |
| 14 | LT | Less then |
| 15 | JMPE | Jump if equal |
| 16 | NOP | No Operation |
| _ | IGL | Illegal action |
